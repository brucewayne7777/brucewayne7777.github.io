This project is a mechatronics-based card shuffler focused on practical hardware implementation, embedded programming, and system integration. It serves as a hands-on engineering project that combines mechanical design, electronics, and C-based control logic into a functional automated device.

### Features

- **Automated card-shuffling mechanism** designed for physical card handling
- **Embedded C implementation** for low-level system behavior and control
- **Hardware-software integration** connecting mechanics, electronics, and logic
- **Prototype-oriented development** with a focus on real-world engineering practice

This project highlights applied mechatronics development through the integration of control, embedded systems, and hardware design in a single prototype.
