# 🌐 brucewayne7777.github.io — Personal Portfolio Website

> Personal academic and engineering portfolio of **Sudhaavan**, Mechatronics student at Berliner Hochschule für Technik (BHT Berlin).

🔗 **Live site:** [https://brucewayne7777.github.io](https://brucewayne7777.github.io)

---

## 👤 About

This is the source repository for Sudhaavan's personal portfolio website, built with **Hugo** and the **HugoBlox (Wowchemy)** academic theme. The site showcases projects, technical skills, and professional experience across mechatronics, robotics, embedded systems, and computer vision.

## 📂 Project Structure

```
brucewayne7777.github.io/
├── config/           # Hugo configuration files
├── content/
│   ├── authors/      # Author profile (bio, skills, social links)
│   ├── home/         # Homepage sections (widgets)
│   ├── project/      # Portfolio projects (one folder per project)
│   └── about/        # About page content
├── assets/           # Custom assets (CSS, images)
├── static/           # Static files served as-is
└── netlify.toml      # Netlify deployment config
```
